API Documentation

ScrapeFlow API

Complete API documentation for integrating ScrapeFlow into your applications

📖 Overview

The ScrapeFlow API provides programmatic access to our powerful web scraping engine. Use it to extract data from any website with our simple RESTful interface.

Base URL

http://localhost:5000

Authentication

Authorization: Bearer YOUR_API_KEY

🔗 API Endpoints

Core endpoints for web scraping operations

GET
/health
Check API server status and health
POST
/scrape
Main scraping endpoint. Extract text, links, images, and metadata from any URL
POST
/batch-scrape
Scrape multiple URLs in a single request
GET
/results/{id}
Retrieve previous scraping results by ID

⚙️ Request Parameters

Parameters for the scraping endpoints

Parameter Type Required Description
url string Yes The URL to scrape. Must be a valid HTTP/HTTPS URL
method string No Scraping method: "requests" or "playwright". Defaults to "requests"
delay number No Delay between requests in milliseconds. Defaults to 1000
extract_links boolean No Extract links from the page. Defaults to true
extract_images boolean No Extract images from the page. Defaults to true
user_agent string No Custom User-Agent string. Defaults to ScrapeFlow bot
timeout number No Request timeout in seconds. Defaults to 30

📤 Response Format

All API responses are returned in JSON format

Success Response

{ "ok": true, "data": { "url": "https://example.com", "title": "Page Title", "text": "Extracted text content...", "word_count": 1234, "characters": 5678, "method": "requests", "links": [ { "href": "https://example.com/page1", "text": "Link text" } ], "images": [ { "src": "https://example.com/image1.jpg", "alt": "Image alt text" } ], "metas": [ { "name": "description", "content": "Page description" } ] } }

Error Response

{ "ok": false, "error": "Failed to scrape URL", "message": "Detailed error message" }

💡 Code Examples

Example implementations in different programming languages

JavaScript / Node.js

const response = await fetch('http://localhost:5000/scrape', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer YOUR_API_KEY' }, body: JSON.stringify({ url: 'https://example.com', method: 'playwright', extract_links: true, extract_images: true }) }); const result = await response.json(); console.log(result.data);

Python

import requests import json response = requests.post('http://localhost:5000/scrape', headers={ 'Content-Type': 'application/json', 'Authorization': 'Bearer YOUR_API_KEY' }, json={ 'url': 'https://example.com', 'method': 'requests', 'extract_links': True, 'extract_images': True } ) result = response.json() print(result['data'])

cURL

curl -X POST http://localhost:5000/scrape \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{ "url": "https://example.com", "method": "requests", "extract_links": true, "extract_images": true }'
